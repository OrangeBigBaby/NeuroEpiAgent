## Scope limitations

- The benchmark remains a draft pending two-expert adjudication.
- SEER clinical analysis is not implemented; only metadata inspection is
  supported.
- CDC WONDER and SEER planning/routing capabilities remain planned or
  infeasible as documented in the capability matrix.
- The NHANES demonstration is a descriptive weighted estimate without
  complex-survey variance; it is not causal, diagnostic, or decision support.

The annotated tag and GitHub Release provide the immutable commit SHA and
release-level maintainer sign-off.
